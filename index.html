How to make public to any private Repo


1. **GitHub par login karein** aur apni repository kholein jo aapko public karni hai.

2. **Settings** tab par click karein (ye option repository ke top menu mein milega).

3. Scroll down karein aur **Danger Zone** section mein "Change repository visibility" option dhoondhein.

4. Wahan par **Change visibility** par click karein.

5. A dialog box open hoga jahan aapko **Make public** ka option milega. Is par click karein.

6. Confirm karne ke liye aapse repository ka naam type karne ko kaha ja sakta hai, to aap repository ka naam type karke confirm kar dein.
